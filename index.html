FAQ

Is this compliant for healthcare-related services?

We add guardrails: no medical diagnosis, no promises. The agent collects info and routes to payment/booking.

Can a human take over the chat?

Yes. MVP can include handoff rules to a manager when needed (optional upgrade).

Can we add more channels later?

Yes—Instagram, web chat, Telegram, etc. MVP starts with one channel to ship fast.
